##### **Proyecto Sprint 7 - Dashboard de Vehículos**



&nbsp;                     **Enlaces del proyecto:**



\- Repositorio GitHub: https://github.com/lermaisaac90-eng/sprint\_7\_project.git

\- Aplicación web en Render: https://sprint-7-project-ld5r.onrender.com



&nbsp;                    **Descripción del proyecto:**



Aplicación web en Streamlit para explorar el dataset vehicles\_us.csv. Muestra un encabezado y dos visualizaciones interactivas: un histograma de odometer y un gráfico de dispersión price vs odometer, activados mediante botones.
